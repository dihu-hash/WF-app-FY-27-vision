# Workforce Mobile App Prototype

A mobile workforce management app built with React and TailwindCSS, featuring time tracking, payroll, scheduling, and a dashboard.

## Features

- Time tracking with functional timer
- Payroll management with visual breakdowns
- Schedule calendar view
- Dashboard with widgets
- Mobile-first design in iPhone 17 mockup

## Setup

```bash
npm install
npm run dev
```

## Tech Stack

- React 18
- Vite
- TailwindCSS
- React Router
- Recharts (for charts)
- React Leaflet (for maps)
- Lucide React (for icons)
